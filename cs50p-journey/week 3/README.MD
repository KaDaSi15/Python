# 📖 Overview

Week 3 was all about making Python programs **robust, reliable, and user-friendly**.

Instead of allowing programs to crash when users enter incorrect input, I learned how to detect errors, handle exceptions gracefully, and provide meaningful feedback. I also explored Python's powerful built-in libraries that make solving problems easier and more efficient.

This week marked an important transition from writing beginner programs to developing applications that behave more like real-world software.

# 🌟 Highlight Projects

## 🏦 ATM Withdrawal System

One of my most advanced projects this week.

This program combines multiple Python concepts into a single real-world application.

### Features

- ✅ Multiple functions
- ✅ Custom exceptions
- ✅ Input validation
- ✅ `try` / `except` / `else`
- ✅ Raising custom errors using `raise`

---

## 🔐 Password Generator

Another highlight of this week was building an interactive password generator.

### Features

- 🔢 Lets the user choose the desired password length.
- 🔀 Generates a strong random password using uppercase letters, lowercase letters, digits, and special characters.
- 🔄 Allows the user to generate a new password repeatedly within the same execution until they are satisfied.
- ✅ Validates user input before generating passwords.
- 🎲 Uses Python's `random` module to create unique passwords each time.

This project strengthened my understanding of loops, input validation, conditional statements, string manipulation, functions, and Python's standard libraries while creating a practical utility that can be used in everyday life.
